How can I build a blog in just one afternoon

17 Apr, 2026 · 1 min read

View on GitHub

yeah,我是无敌的,七小时前想打造一款个人blog，很快就撸出来了。

虽然自己是联系两年半的编程老手，对这种纯前端blog网站按理说应该手拿把捏吧。实际上还是有些难度的,踩过坑的，特来分享下经验。

前置条件

  • codex（i hate cc,a/）
  • prompt
  • github
  • chrome mcp skills
  • 一个参考博客（如https://github.com/steipete/steipete.me）

过程

踩坑图：开始还想着让ai浏览器截图一张一张学，后面发现学个p，直接对着抄就完事了。提示词加上https://github.com/steipete/steipete.me 直接对着抄！！！

感言
